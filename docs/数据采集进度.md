## 文件命名规范

```
data/raw/
  monthly/
    macro_YYYYMMDD.xlsx       # B1：月频宏观大包（当次导出日期）
    bench_YYYYMMDD.xlsx       # B5：月频基准序列
  daily/
    rates_YYYYMMDD.xlsx       # B2：日频利率包
    market_YYYYMMDD.xlsx      # B3：日频行情包
    fred_YYYYMMDD.csv         # B4：T5YIE
  event/
    cb_events_YYYYMMDD.xlsx   # B6：央行事件
  news/
    news_YYYYMMDD.xlsx        # B7：新闻情绪
```

> `YYYYMMDD` 为文件**导出日期**（非数据截止日期），例如 `macro_20260309.xlsx`。
> 数据截止日期在 ETL 入库时从文件内容自动识别，不放在文件名中。

## 月度操作 SOP（每月更新流程）

```
1. 月初等统计局数据发布（通常次月15日前）
2. 打开 Choice → 批量导出 B1（月频宏观）→ 存 data/raw/monthly/macro_YYYYMMDD.xlsx
3. 同时补充 B2 + B3（日频历史补齐至月末）→ 存 data/raw/daily/
4. B4 T5YIE 可脚本自动拉取（或手工从 FRED 下载）
5. B5 月频基准序列按月补充
6. B6 事件日历按实际发生时间追加（不定期）
7. 运行 ETL：
     python -m src.etl.load_macro_monthly --file data/raw/monthly/macro_YYYYMMDD.xlsx
     python -m src.etl.load_macro_daily   --file data/raw/daily/rates_YYYYMMDD.xlsx
     python -m src.etl.load_market_daily  --file data/raw/daily/market_YYYYMMDD.xlsx
8. 构建快照：
     python -m src.aggregation.build_snapshot --month YYYY-MM
```
