## 文件命名规范

```
data/raw/
  monthly/
    nbs_macro_core_YYYYMMDD.xlsx     # B1a：月频宏观核心（NBS，PBOC，MOF）
    nbs_macro_noncore_YYYYMMDD.xlsx  # B1b：月频宏观非核心（NBS，海关总署）
    bench_YYYYMMDD.xlsx              # B5：月频基准序列
  daily/
    rates_YYYYMMDD.xlsx              # B2：日频利率包
    market_YYYYMMDD.xlsx             # B3：日频行情包
    fred_YYYYMMDD.csv                # B4：T5YIE
  event/
    cb_events_YYYYMMDD.xlsx          # B6：央行事件
  news/
    news_YYYYMMDD.xlsx               # B7：新闻情绪
```

> `YYYYMMDD` 为文件**导出日期**（非数据截止日期）。
> 命名前缀可含来源标识（如 `nbs_`），便于快速识别；数据截止日期从文件内容自动识别，不放入文件名。

## 月度操作 SOP（每月更新流程）

```
1. 月初等统计局数据发布（通常次月15日前）
2. Choice 导出 B1a → data/raw/monthly/nbs_macro_core_YYYYMMDD.xlsx
3. Choice 导出 B1b → data/raw/monthly/nbs_macro_noncore_YYYYMMDD.xlsx
   ⚠️  确认 REAL_GDP_QOQ 是否已补充（季度数据，单独导出）
4. Choice 导出 B2 → data/raw/daily/rates_YYYYMMDD.xlsx（补齐至月末）
5. Choice 导出 B3 → data/raw/daily/market_YYYYMMDD.xlsx（补齐至月末）
6. B4 T5YIE 可脚本自动拉取（或手工从 FRED 下载）→ data/raw/daily/fred_YYYYMMDD.csv
7. B5 月频基准序列按月补充 → data/raw/monthly/bench_YYYYMMDD.xlsx
8. B6 事件日历按实际发生时间追加（不定期）
9. 运行 ETL：
     python -m src.etl.load_macro_monthly --file data/raw/monthly/nbs_macro_core_YYYYMMDD.xlsx
     python -m src.etl.load_macro_monthly --file data/raw/monthly/nbs_macro_noncore_YYYYMMDD.xlsx
     python -m src.etl.load_macro_daily   --file data/raw/daily/rates_YYYYMMDD.xlsx
     python -m src.etl.load_market_daily  --file data/raw/daily/market_YYYYMMDD.xlsx
10. 构建快照：
     python -m src.aggregation.build_snapshot --month YYYY-MM
```
