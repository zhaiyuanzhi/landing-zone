# 市场分析与资产配置报告结构


| 标题       | 必要性 | 核心内容        | 必要性 | 包含要素                                                                                                                                                                                                                                                                                                |
| -------- | --- | ----------- | --- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 引言       | 非必要 | 分析背景        | 非必要 | 简练复述客户问题，并肯定客户，一句话总结当前市场的核心表现或矛盾。                                                                                                                                                                                                                                                                   |
| 整体市场结构   | 必要  | 宏观周期定位      | 必要  | **必要要素**：当前经济周期阶段，并提供当前市场处于复苏/过热/滞胀/衰退哪个象限。 **非必要要素**：对库存周期（工业企业产品成品库存、PPI与CPI剪刀差）、信用周期（社融增速、M2与M1剪刀差、信贷脉冲等）、技术长波（全要素生产率、新兴产业等）进行描述。 **扩展性**：利用模型能力，对判断当前宏观周期定位的数据、推导逻辑、判断结论均可展示。                                                                                                                   |
| 整体市场结构   | 必要  | 市场动能拆解      | 非必要 | **必要要素**：对消费（社会消费品零售总额、居民储蓄率、消费者信心指数、服务消费占比等）、投资（固定资产投资〔基建/制造/房地产三分法〕、民间投资增速、设备购置投资）、出口（出口增速、PMI新出口订单、出口价格指数、市场份额变化）进行拆解。 **非必要要素**：产业景气扫描，对上游（CRB指数、南华工业品指数、油价/铜价/铁矿石价格）、中游（发电量、货运量、挖掘机开工小时数、水泥发运率）、下游（30城商品房成交面积、汽车销量、家电出货）进行描述。 **扩展性**：利用模型能力，对判断当前市场动能拆解的数据、推导逻辑、判断结论均可展示。                     |
| 整体市场结构   | 必要  | 货币政策与财政政策分析 | 非必要 | **必要要素**：央行操作、央行季度报告、财政政策、财政政策。 **非必要要素**：政策利率（MLF/OMO/LPR）、逆回购操作量、同业存单利率、票据利率、美元指数、中美利差、外汇储备、一般公共预算收入/支出增速、税收结构变化（反映经济质量）、赤字调整、专项债发行进度（观察“提前批”节奏）、城投债净融资、政策性金融工具投放、重大项目开工率、基建到位资金、PPP新入库项目。 **扩展性**：利用模型能力，其他同类要素均可展示。                                                                             |
| 整体市场结构   | 必要  | 国际环境分析      | 非必要 | **必要要素**：美联储政策、重大地缘政治事件、大宗商品价格变动、跨境资金流动。 **非必要要素**：美联储主席讲话、布伦特原油、LME铜、黄金价格、北向资金、外汇占款、外汇储备。 **扩展性**：利用模型能力，其他同类要素均可展示。                                                                                                                                                                               |
| 整体市场结构   | 必要  | 风险预警分析      | 必要  | **必要要素**：根据上述内容判断市场存在的系统性风险和坏情况。                                                                                                                                                                                                                                                                    |
| 大类资产驱动因素 | 必要  | 股票类分析       | 必要  | **必要要素**：股票类市场事件及表现（东方财富网资讯）。 **非必要要素**：客户股票类持仓明细、客户股票类持仓近1月收益情况。 **扩展性**：利用模型能力，其他同类要素均可展示。                                                                                                                                                                                                        |
| 大类资产驱动因素 | 必要  | 固收类分析       | 必要  | **必要要素**：固收类市场事件及表现（东方财富网资讯）。 **非必要要素**：客户固收类持仓明细、客户固收类持仓近1月收益情况。 **扩展性**：利用模型能力，其他同类要素均可展示。                                                                                                                                                                                                        |
| 大类资产驱动因素 | 必要  | 另类分析        | 必要  | **必要要素**：另类市场事件及表现（东方财富网资讯）。 **非必要要素**：客户另类持仓明细、客户另类持仓近1月收益情况。 **扩展性**：利用模型能力，其他同类要素均可展示。                                                                                                                                                                                                           |
| 大类资产驱动因素 | 必要  | 现金类分析       | 必要  | **必要要素**：现金类市场事件及表现（东方财富网资讯）。 **非必要要素**：客户现金类持仓明细、客户现金类持仓近1月收益情况。 **扩展性**：利用模型能力，其他同类要素均可展示。                                                                                                                                                                                                        |
| 后续关注方向   | 必要  | 宏观环境追踪      | 非必要 | **必要要素**：投顾月度宏观观点。 **非必要要素**：社融增速、制造业PMI新订单指数、地产销售面积、CPI/PPI剪刀差、出口增速（美元计价）、货币政策、财政政策、产业政策。 **扩展性**：利用模型能力，其他同类要素均可展示。                                                                                                                                                                               |
| 后续关注方向   | 必要  | 大类资产调整      | 必要  | **必要要素**：客户风险等级、客户近1月产品浏览记录、投顾月度大类资产观点（财管-市场回顾、投研-大回访建议书）。 **非必要要素**： 1）客户大类持仓低缺配情况。 2）权益资产当前市场阶段（红利指数 vs 科创50比值、ERP分位数、铜等有色金属价格、白酒批价、PMI订单、期限利差、M1-M2、信用利差、失业率）。 3）固收资产当前阶段（当前10Y国债收益率、城投债/产业债资讯、AA级信用利差）。 4）另类资产当前阶段（实际利率〔10Y TIPS〕、央行购金量、南华商品指数波动率、中美利差、汇率预期、美元存款利率）。 **扩展性**：利用模型能力，其他同类要素均可展示。 |
| 后续关注方向   | 必要  | 风险提示        | 必要  | **必要要素**：各大类市场表现不及预期的风险归因与预警。 **非必要要素**：流动性风险、信用风险、汇率风险、地缘政治风险、政策监管风险。 **扩展性**：利用模型能力，其他同类要素均可展示。                                                                                                                                                                                                   |
