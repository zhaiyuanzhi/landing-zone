# 单独汇总：需加工的原始指标 → 加工方式 → 最终喂给模型指标

这一节只单独汇总两类：

1. 原始日度指标，需先转成月度后再进入模型输入。
2. 专项债相关指标，需先基于原始指标加工派生后，再作为最终输入指标。

### 1）日度转月度：原始指标 → 月频口径 → 原因 → 最终喂给模型指标

| 类型    | 原始指标代码             | 原始指标名称            | 月频口径 | 原因 | 是否改名 | 最终喂给模型指标         |
| ----- | ------------------ | ----------------- | ---- | ---- | ---- | ---------------- |
| 日度转月度 | BRENT_CRUDE        | 期货收盘价(连续):ICE布油   | 月末值 | 属于市场价格类指标，月末值更能反映当月结束时的最新市场定价状态，适合做宏观与市场状态判断。 | 否    | BRENT_CRUDE      |
| 日度转月度 | XAUUSD             | 期货收盘价(连续):COMEX黄金 | 月末值 | 属于市场价格类指标，月末值比月均值更能体现当前时点的金价水平与市场风险偏好变化。 | 否    | XAUUSD           |
| 日度转月度 | DR007              | DR007             | 月均值 | 属于短端资金利率，受月末、季末、节假日和监管考核扰动较大，单一月末值噪声偏强，月均值更能代表当月整体资金面松紧。 | 否    | DR007            |
| 日度转月度 | AA_CREDIT_YIELD_3Y | AA级信用债收益率（3年）     | 月末值 | 属于信用债收益率水平指标，月末值更适合表达当月结束时信用定价与融资条件所处位置。 | 是    | CREDIT_YTM_AA_3Y |
| 日度转月度 | CGB_3Y             | 中债国债到期收益率:3年      | 月末值 | 属于国债收益率水平指标，月末值更适合反映月末利率曲线状态与债券市场定价水平。 | 是    | CGB_3Y_YTM       |

说明：

- 这几类指标都不能直接把原始日频序列喂给模型，而应先统一整理成月频。
- 月末值适用于价格类、收益率水平类指标，用于保留“当月结束时市场处在什么位置”的状态信息。
- 月均值适用于短端资金利率这类高波动、易受时点扰动的指标，用于降低月末噪声，更稳定地表示当月整体环境。


### 2）专项债加工：原始指标 → 加工方式 → 最终喂给模型指标


| 类型    | 原始指标代码                                                    | 原始指标名称                    | 必要加工                                                        | 是否改名 | 最终喂给模型指标              | 备注          |
| ----- | --------------------------------------------------------- | ------------------------- | ----------------------------------------------------------- | ---- | --------------------- | ----------- |
| 专项债加工 | SPECIAL_BOND_ISSUE_CUM + LOCAL_SPECIAL_BOND_TARGET_ANNUAL | 新增专项债券:累计值 + 政府预期目标:地方专项债 | `SPECIAL_BOND_ISSUE_CUM / LOCAL_SPECIAL_BOND_TARGET_ANNUAL` | 是    | SPECIAL_BOND_PROGRESS | 适合作为最终月度主指标 |


补充说明：

- `SPECIAL_BOND_ISSUE_CUM` 本身是原始累计值，但保留它有意义，因为它能补充专项债发行的绝对量级信息。
- `SPECIAL_BOND_PROGRESS` 是基于 `SPECIAL_BOND_ISSUE_CUM` 和 `LOCAL_SPECIAL_BOND_TARGET_ANNUAL` 加工得到的最终指标，用于表示年度执行进度。
